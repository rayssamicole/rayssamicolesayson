My Works




Branding

​  This project is about creating an advertisement for a specific brand. In South Korea, Chilsung cider is a well-known flavored carbonated beverage. The purpose of this project is to highlight the artist's creativity while also building relationships with the target audience. To attract the target audience, the artist uses bright colors and characters.


3d Modeling

 This project is a 3d modeling project made using Blender software. The concept of the project is Minimalist-Mediterranean kitchen. With simple and romantic Southern European aesthetics: light and neutral tones in color scheme with an accent of "Reseda Green," as well as extensive use of natural materials such as ceramics, wood, marble, and sodalite.
